协商缓存和强制缓存？

强制缓存的优先级高于协商缓存。如果强制缓存有效，浏览器不会进行协商缓存的请求。

在什么情况下使用强制缓存？什么情况下使用协商缓存？​

解答：

强制缓存适用于不经常变动的资源，如图片、CSS、JavaScript 等。 协商缓存适用于可能会频繁变动的资源，如 HTML 页面、API 接口等。

如何实现强制缓存？​

可以通过设置 Cache-Control 或 Expires 头部来实现强制缓存。例如：

Cache-Control: max-age=3600
Expires: Wed, 21 Oct 2025 07:28:00 GMT

如何实现协商缓存？​

可以通过设置 Last-Modified 和 If-Modified-Since 或 ETag 和 If-None-Match 头部来实现协商缓存。例如：

Last-Modified: Wed, 21 Oct 2023 07:28:00 GMT
或
ETag: "686897696a7c876b7e"
