Vue 生态

更多其他框架和库​

Element UI​

Element UI 是一套为开发者、设计师和产品经理准备的基于 Vue 2.0 和 Vue 3.0 的桌面端组件库。它提供了丰富的 UI 组件，如按钮、表格、表单、对话框等，设计风格简洁大方，易于使用和定制

Ant Design Vue​

Ant Design Vue 是 Ant Design 的 Vue.js 实现，专为企业级后台产品设计。它提供了高质量的 UI 组件，遵循 Ant Design 的设计语言，具有响应式布局和良好的文档支持

Vuetify​

Vuetify 是一个基于 Material Design 的 Vue.js 组件库。它提供了美观且易于使用的 UI 组件，支持响应式布局和主题定制，适合现代化的设计风格

  • Vuetify 3 支持 IE11 吗？

    不支持。如果您需要支持 IE11，请使用 Vuetify 2。

Quasar​

Quasar 是一个全面的 Vue.js 框架，提供了跨平台的 UI 组件和工具。它支持构建桌面和移动应用程序，组件库丰富，包括按钮、图标、菜单等

Bootstrap Vue​

Bootstrap Vue 是基于 Bootstrap 的 Vue.js 组件库，提供了与 Bootstrap 样式和布局系统无缝集成的 UI 组件。

iView​

iView 是一套基于 Vue.js 的桌面端组件库，提供了美观且易于使用的 UI 组件。

Mint UI​

Mint UI 是一套基于 Vue.js 的移动端组件库，提供了简洁、易于使用的 UI 组件。

Vant​

Vant 是有赞前端团队开源的移动端组件库，提供了 60 多个高质量的组件，支持 Vue 2 和 Vue 3。

NutUI​

NutUI 是京东风格的 Vue 移动端组件库，提供了 70 多个高质量的组件，支持按需引用、主题定制和 TypeScript。

BalmUI​

BalmUI 是为 Vue.js 3.0 量身订制的模块化且高可定制化的 Material Design UI 库。

Vuestic UI​

Vuestic UI 是 Vue 最漂亮的开源管理面板之一，提供了可维护的 Vue 代码和灵活的组件。

IDUX UI​

IDUX UI 是一套企业级中后台 UI 组件库，致力于提供高效愉悦的开发体验。它基于 Vue 3.x 和 TypeScript 开发，支持国际化语言和灵活的主题定制

RelaxPlus​

RelaxPlus 是一套为开发者学习 Vue 3.x 而准备的友好、简洁、轻盈、精致的桌面端组件库。它适合需要快速构建桌面端应用程序的项目

Viewer Design​

Viewer Design 是一款基于 Vue3.0 和 TypeScript 开发的中后台 UI 组件库，提供了高配置性的组件和更好的交互体验。它适合需要高度定制化和高质量组件的项目

Vexip UI​

Vexip UI 提供了一系列开箱即用的组件，使用全新的 Vue 3.0 组合式 API 编写。
