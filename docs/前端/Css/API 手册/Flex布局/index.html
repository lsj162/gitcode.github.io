Flex 布局

Flexbox 布局？

CSS Flexbox（Flexible Box Layout）是一种用于在容器中分配空间并对齐内容的布局模式。 Flexbox 提供了一组强大的属性，可以控制容器（父元素）和项目（子元素）的布局行为。

容器属性（Grid Container Properties）​

  • 容器属性：定义布局方向、换行、对齐方式等。

display 用途：定义一个 flex 容器。

  • flex：块级 flex 容器。
  • inline-flex：行内级 flex 容器。

flex-direction 用途：定义主轴方向，即 flex 项目排列的方向。

  • row（默认值）：水平方向，从左到右。
  • row-reverse：水平方向，从右到左。
  • column：垂直方向，从上到下。
  • column-reverse：垂直方向，从下到上。

flex-wrap 用途：定义 flex 项目是否换行。

  • nowrap（默认值）：不换行，所有项目在一行或一列中。
  • wrap：换行，项目将从上到下或从左到右换行。
  • wrap-reverse：换行，项目将从下到上或从右到左换行。

flex-flow 用途：是 flex-direction 和 flex-wrap 的简写属性。

  • <flex-direction> <flex-wrap>

justify-content 用途：定义项目在主轴上的对齐方式。

  • flex-start（默认值）：项目对齐到主轴起点。
  • flex-end：项目对齐到主轴终点。
  • center：项目在主轴上居中对齐。
  • space-between：项目之间均匀分布，首尾项目对齐到两端。
  • space-around：项目之间均匀分布，首尾项目周围有空间。
  • space-evenly：项目之间以及与两端之间有均匀的空间。

align-items 用途：定义项目在交叉轴上的对齐方式。

  • stretch（默认值）：项目拉伸以填满容器。
  • flex-start：项目对齐到交叉轴起点。
  • flex-end：项目对齐到交叉轴终点。
  • center：项目在交叉轴上居中对齐。
  • baseline：项目根据它们的基线对齐。

align-content 用途：定义多行项目在交叉轴上的对齐方式（仅在多行时有效）。

  • stretch（默认值）：多行项目拉伸以填满容器。
  • flex-start：多行项目对齐到交叉轴起点。
  • flex-end：多行项目对齐到交叉轴终点。
  • center：多行项目在交叉轴上居中对齐。
  • space-between：多行项目之间均匀分布，首尾项目对齐到两端。
  • space-around：多行项目之间均匀分布，首尾项目周围有空间。
  • space-evenly：多行项目之间以及与两端之间有均匀的空间。

gap, row-gap, column-gap 用途：定义 flex 项目之间的间距。

  • 长度单位（如 px, em, %）

项目属性（Flex Item Properties）​

  • 项目属性：控制项目的顺序、大小、对齐方式等。

flex-grow

用途：定义 flex 项目的放大比例。

  • 数字（默认为 0）

flex-shrink 用途：定义 flex 项目的缩小比例。

  • 数字（默认为 1）

flex-basis 用途：定义在分配多余空间之前，项目在主轴方向上的初始大小。

  • 长度值（如 px, %, em）或 auto（默认为 auto）

flex 用途：是 flex-grow, flex-shrink 和 flex-basis 的简写属性。

  • flex-grow flex-shrink flex-basis

align-self 用途：定义单个 flex 项目在交叉轴上的对齐方式，覆盖容器的 align-items 属性。

  • auto（默认值）：继承容器的 align-items。
  • stretch：拉伸以填满容器。
  • flex-start：对齐到交叉轴起点。
  • flex-end：对齐到交叉轴终点。
  • center：居中对齐。
  • baseline：根据基线对齐。

order 用途：定义 flex 项目的排列顺序。

  • 整数（默认为 0）
