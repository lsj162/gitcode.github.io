Linux 文件操作命令

在 Linux 系统下，文件操作常用的命令包括：

  1. ls：列出目录内容。

    ls [选项] [目录名]

  2. cd：更改目录。

    cd [目录名]

  3. pwd：显示当前工作目录的路径。

    pwd

  4. mkdir：创建新目录。

    mkdir [目录名]

  5. touch：创建新文件或更新现有文件的时间戳。

    touch [文件名]

  6. cp：复制文件或目录。

    cp [源文件或目录] [目标文件或目录]

  7. mv：移动文件或目录，或重命名文件或目录。

    mv [源文件或目录] [目标文件或目录]

  8. rm：删除文件或目录。

    rm [选项] [文件或目录] ❗️ 一定要指定文件目录，防止删除重要文件

  9. rmdir：删除空目录。

    rmdir [目录名]

  10. cat：显示文件内容。

    cat [文件名]

  11. more：分页显示文件内容。

    more [文件名]

  12. less：分页显示文件内容，支持向前和向后浏览。

    less [文件名]

  13. head：显示文件的前几行。

    head [文件名]

  14. tail：显示文件的最后几行。

    tail [文件名]

  15. chmod：更改文件或目录的权限。

    chmod [选项] [权限] [文件或目录]

  16. chown：更改文件或目录的所有者和/或组。

    chown [选项] [所有者]:[组] [文件或目录]

  17. find：在文件系统中搜索文件。

    find [路径] [选项] [表达式]

  18. grep：在文件中搜索特定的文本模式。

    grep [选项] '搜索模式' [文件名]

这些命令是 Linux 系统中进行文件操作的基础，掌握它们对于日常的 Linux 使用非常重要。
