Linux系统命令

查看系统磁盘占用​

df -hl

查看端口​

sudo netstat -tuln || sudo ss -tulnp
# -t 是 TCP，-u 是 UDP，-l 是监听中，-n 是显示数字格式的端口和地址 -p 是pid信息

查看某个端口的使用情况​

sudo lsof -i :端口号

查看具体服务绑定的端口​

ps aux | grep 服务名
