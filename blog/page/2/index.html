使用 VPN 或类似的翻墙工具在我国是一个法律灰区，某些情况下可能会涉及法律风险。

那么当企业必须需要时​

企业如果需要使用 VPN 来进行跨境业务、远程办公等，可以通过合法途径申请和使用。根据我国的法律法规，企业使用 VPN 必须满足相关要求，并通过政府批准的正规渠道来获得许可。以下是企业申请 VPN 的一些基本步骤：

  1. 了解相关法律法规：

    • 根据我国的《网络安全法》及其他相关法律，企业使用 VPN 必须符合法律规定。未经许可的 VPN 连接可能会导致法律风险。
    • 企业使用 VPN 主要用于合规的商业用途，例如跨境数据传输、远程办公等。

  2. 选择合规的 VPN 服务提供商：

    • 企业可以选择与我国政府批准的 VPN 服务提供商合作，这些服务商需要获得相关部门的授权和备案。一般来说，国内运营商如我国电信、我国联通、我国移动等也提供合法的企业级 VPN 服务。
    • 这些合规的 VPN 服务商会提供符合我国法规的专线 VPN 或云 VPN 服务。

  3. 向相关政府部门备案：

    • 企业若需要使用跨境 VPN，需要向相关部门进行备案。具体可以联系当地的工信部门（如省级通信管理局）或者国家互联网信息办公室（CAC）进行咨询和备案。
    • 企业需要提供业务用途、数据传输类型、网络安全保障措施等详细信息。

  4. 审查与审批：

    • 在一些情况下，企业可能需要接受政府的审查或审批，确保其使用 VPN 服务的方式不违反国家的安全和监管政策。

  5. 签订相关合同和协议：

    • 合法的 VPN 服务商和企业之间会签订正式的服务协议，明确双方的责任、服务内容以及法律合规要求。

  6. 加强网络安全管理：

    • 企业还需要加强网络安全管理，确保通过 VPN 进行的跨境通信和数据传输符合我国的网络安全要求。包括对敏感数据的加密、访问权限控制等措施。

如果企业没有合法的 VPN 许可而仍使用未经批准的 VPN 服务，可能会面临罚款、警告或更严重的法律后果。为了确保合规，建议企业在申请 VPN 服务时，咨询法律专业人士并与合规的服务商合作。

什么是重绘（Repaint）？​

只改变元素的外观，不影响布局。

  • 改变元素的颜色、背景色、透明度等外观属性。

什么是回流（Reflow）？​

改变元素的几何属性或布局，影响页面的整体布局。

  • 改变元素的几何属性
  • 改变元素的布局属性（如 display, position, float, flex 等）。
  • 添加或删除 DOM 元素。
  • 改变浏览器窗口的大小。
  • 改变字体大小。

如何减少重绘和回流的次数？​

  • 批量修改样式
  • 使用文档片段
  • 使用 transform 和 opacity 属性
  • 虚拟 DOM
  • requestAnimationFrame
  • 使用 CSS3 硬件加速：使用 transform, opacity, filters
