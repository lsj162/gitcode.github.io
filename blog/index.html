  1. 构建渲染树（Render Tree Construction）

  • 结合 DOM 和 CSSOM：将 DOM 树和 CSSOM 树结合，生成渲染树（Render Tree）。

  • 执行 JavaScript：当浏览器遇到<script>标签时，会暂停 HTML 解析，执行 JavaScript 代码。
  • JavaScript 可以修改 DOM 和 CSSOM，从而触发重新解析、布局和绘制。

重绘回流防抖？

2025年3月26日 · 阅读需 1 分钟

什么是重绘（Repaint）？​

只改变元素的外观，不影响布局。

  • 改变元素的颜色、背景色、透明度等外观属性。

什么是回流（Reflow）？​

改变元素的几何属性或布局，影响页面的整体布局。

  • 改变元素的几何属性
  • 改变元素的布局属性（如 display, position, float, flex 等）。
  • 添加或删除 DOM 元素。
  • 改变浏览器窗口的大小。
  • 改变字体大小。

如何减少重绘和回流的次数？​

  • 批量修改样式
  • 使用文档片段
  • 使用 transform 和 opacity 属性
  • 虚拟 DOM
  • requestAnimationFrame
  • 使用 CSS3 硬件加速：使用 transform, opacity, filters
