常见的硬件知识​

涵盖了计算机系统中各种物理组件的理解和操作。以下是一些常见的硬件知识：

  1. 中央处理器（CPU）：

    • 理解 CPU 的架构和工作原理。
    • 不同制造商的 CPU，如 Intel 和 AMD。

  2. 内存（RAM）：

    • 内存类型和速度。
    • 内存的工作原理和作用。

  3. 存储设备：

    • 硬盘驱动器（HDD）和固态驱动器（SSD）的区别。
    • 存储容量和读写速度。
    • 光盘驱动器和闪存驱动器。

  4. 主板：

    • 主板的组成和功能。
    • 插槽和接口，如 PCI Express、RAM 插槽等。

  5. 显卡（GPU）：

    • 显卡的作用和性能。
    • 图形处理单元（GPU）的概念。

  6. 电源：

    • 计算机电源单元的功能。
    • 不同电源规格和连接器。

  7. 显示器：

    • 不同类型的显示器技术，如液晶显示器（LCD）、有机发光二极管（OLED）等。
    • 分辨率和刷新率的概念。

  8. 输入设备：

    • 键盘、鼠标和触摸板的操作和特性。
    • 不同类型的输入设备，如扫描仪和数字板。

  9. 网络适配器：

    • 网络接口卡（NIC）的作用。
    • 无线网络标准，如 Wi-Fi。

  10. 外部接口：

    • USB、HDMI、VGA 等接口的作用和特性。
    • 数据传输速度和连接标准。

  11. 散热系统：

    • 散热风扇和散热片的作用。
    • 液冷系统的原理。

  12. BIOS/UEFI：

    • 基本输入/输出系统（BIOS）或统一可扩展固件接口（UEFI）的作用。
    • 固件更新和设置。

这些硬件知识是理解和管理计算机系统的基础。对于计算机专业或 IT 领域的从业者，深入了解这些硬件知识是非常重要的。

目前主流的系统​

截至我知识截止日期（2022 年 1 月），以下是一些主流的操作系统。请注意，技术领域不断发展，可能会有新的系统推出或者现有系统发生变化。以下列出的系统可能在现在仍然是主流，但也可能存在变化。

  1. Windows 系列：

    • Windows 10 是微软公司最新的主流桌面操作系统。
    • Windows Server 系列用于服务器环境。

  2. macOS：

    • macOS 是苹果公司的桌面和笔记本电脑操作系统，用于 Macintosh 计算机。

  3. Linux 发行版：

    • Linux 是一种开源操作系统内核，有许多不同的发行版，如 Ubuntu、Fedora、Debian、CentOS 等，用于各种应用，包括桌面、服务器和嵌入式系统。

  4. Android：

    • Android 是谷歌开发的移动设备操作系统，主要用于智能手机和平板电脑。

  5. iOS：

    • iOS 是苹果公司开发的移动设备操作系统，主要用于 iPhone、iPad 和 iPod Touch。

  6. Chrome OS：

    • Chrome OS 是由谷歌开发的操作系统，设计用于 Chromebook 等基于云的轻量级计算设备。

  7. BSD 系列：

    • 包括 FreeBSD、OpenBSD、NetBSD 等，它们是基于 BSD（伯克利软件发行版）的开源操作系统。

  8. Unix：

    • Unix 是一种多用户、多任务操作系统，有多个衍生版本，如 AIX、HP-UX、Solaris 等。

请注意，技术行业变化迅速，新的操作系统可能已经推出，而某些系统可能在主流市场上失去影响力。确保查阅最新的资讯以获取有关当前主流操作系统的准确信息。

什么是计算机网络?

2025年2月3日 · 阅读需 1 分钟

计算机网络是指将多台计算机和其他网络设备连接在一起，以便它们可以互相通信和共享资源的系统。这些设备可以通过有线或无线连接相互链接，以便进行数据传输、通信和资源共享。

::: tip 计算机网络使用不同的协议和技术来管理数据传输、确保数据安全、处理网络拓扑、解决网络故障等。它们也涉及网络硬件设备，如路由器、交换机、防火墙，以及网络软件，如操作系统、网络协议栈和应用程序。 :::

它在现代社会和工作中扮演着至关重要的角色，已成为信息时代的基础设施之一。

评论

2025年2月3日 · 阅读需 2 分钟

使用 giscus 实现评论功能。https://giscus.app

完成上面三步骤，即可验证。

安装​

pnpm add @giscus/vue

配置​

<!-- /components/Comments.vue -->

<template>
  <Giscus
    v-if="showComment"
    :repo="giscusConfig.repo"
    :repo-id="giscusConfig.repo_id"
    :category="giscusConfig.category"
    :category-id="giscusConfig.category_id"
    mapping="title"
    strict="0"
    reactions-enabled="1"
    emit-metadata="0"
    input-position="top"
    theme="light"
    lang="zh-CN"
    loading="lazy"
    crossorigin="anonymous"
    async
  />
</template>

<script lang="ts" setup>
import Giscus from "@giscus/vue";
import { useRoute } from "vitepress";
import { reactive, ref, watch } from "vue";
const route = useRoute();

const giscusConfig = reactive({
  repo: "...", // # 这里就是giscus官方提供的
  repo_id: "...", // # 这里就是giscus官方提供的
  category: "...", // # 这里就是giscus官方提供的
  category_id: "...", // # 这里就是giscus官方提供的
});

const showComment = ref(true);

watch(
  () => route.path,
  () => {
    showComment.value = false;

    setTimeout(() => {
      showComment.value = true;
    }, 200);
  },
  {
    immediate: true,
  }
);
</script>

添加到全局​

// https://vitepress.dev/guide/custom-theme
import Theme from "vitepress/theme";
import { h } from "vue";
import Comments from "../components/Comments.vue";

export default {
  extends: Theme,

  Layout: () => {
    return h(Theme.Layout, null, {
      // https://vitepress.dev/guide/extending-default-theme#layout-slots
      "doc-after": () => h(Comments),
    });
  },
};
